# Code of conduct

GroupBridge follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, assume good intent, focus criticism on the work, and do not harass or
discriminate. Maintainers may remove content or participation that makes the project
unsafe or unwelcoming. Report conduct concerns privately through the repository owner's
GitHub profile contact options; do not use public issues for sensitive reports.
