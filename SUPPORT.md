# Support

GroupBridge is community-maintained and provided without warranty.

- Use GitHub Discussions for setup questions and design conversations when enabled.
- Use the structured issue forms for reproducible bugs and focused feature requests.
- Use private vulnerability reporting for anything that could expose credentials,
  grant unauthorized access, forge events, or prune the wrong membership.

Before asking for help, run `make verify` or `make demo-status`, inspect the controller
logs, and include the GroupBridge, Keycloak, GitLab, Kubernetes, k3d/Helm versions plus a
minimal configuration with all secrets and identity data removed.
